# Task Manager - Frontend (Angular 19)

A modern, responsive **Single Page Application (SPA)** built with **Angular 19**, providing a seamless and interactive user experience for managing daily tasks.

---

## Technical Features

- **Standalone Components:** Leveraging the latest Angular architecture for reduced boilerplate and faster builds.
- **RxJS State Management:** Reactive handling of task streams and asynchronous data flow.
- **HTTP Interceptors:** Automated attachment of **JWT Bearer Tokens** to every outgoing request.
- **Route Guards:** Robust **AuthGuards** to prevent unauthorized access to the dashboard.

---

## User Experience (UX)

- **Live TimeAgo Pipe:** Custom pipe to display creation dates in a human-readable format (e.g., "5 minutes ago") with real-time updates.
- **Instant UI Updates:** Tasks added via `unshift()` appear at the top of the list immediately without a page refresh.
- **Responsive Design:** Styled with **Bootstrap 5** to ensure full compatibility across mobile, tablet, and desktop.
- **Dynamic Badges:** Visual indicators for task status (Completed, Active, or Soft-Deleted).
